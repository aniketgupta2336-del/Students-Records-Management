# 🎓 Student Records Management System

A web-based Student Records Management System developed using **Python Django Framework** and **SQLite Database**. The application helps educational institutions manage student information digitally by providing functionalities such as adding, viewing, updating, searching, and deleting student records.

## 🚀 Features

* User Authentication (Login/Logout)
* Add New Student Records
* View Student Details
* Search Student Records
* Update Student Information
* Delete Student Records
* SQLite Database Integration
* Responsive User Interface

## 🛠 Technologies Used

* Python
* Django
* HTML
* CSS
* Bootstrap
* SQLite
* Visual Studio Code

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/student-records-management.git
```

2. Navigate to project directory

```bash
cd student-records-management
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Apply migrations

```bash
python manage.py migrate
```

5. Run the server

```bash
python manage.py runserver
```

6. Open browser

```text
http://127.0.0.1:8000/
```
